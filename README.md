# EduGenie - AI-Powered Educational Assistant

EduGenie is a lightweight, responsive web application designed to simplify learning. It combines cloud-based Large Language Models (**Google Gemini 3.5**) with local transformers (**LaMini-Flan-T5-77M**) running on the CPU to deliver hybrid AI capabilities.
